## Contoh Penggunaan

### Tambah Karyawan Baru

```
1. Klik "Tambah Karyawan"
2. Isi data:
   - Nama: John Doe
   - Posisi: Software Engineer
   - Departemen: IT
   - Status: Aktif
   - Telepon: 08123456789
   - Email: john@example.com
   - Gaji: 10000000
   - Tanggal Bergabung: 01/12/2024
3. Tambah Job Description (opsional)
4. Klik "Simpan"
```

### Filter Data

```
- Status: Pilih "Aktif" untuk melihat karyawan aktif saja
- Departemen: Pilih departemen tertentu
- Search: Ketik nama/posisi/telepon untuk mencari
- Klik "Filter"
```

### Export Data

```
- PDF: Klik icon merah (PDF) untuk download laporan PDF
- Excel: Klik icon hijau (Excel) untuk download file Excel
```

## Status Karyawan

-   **Aktif** (hijau): Karyawan masih bekerja
-   **Tidak Aktif** (kuning): Karyawan cuti/tidak aktif sementara
-   **Resign** (merah): Karyawan sudah resign

## Tips

1. **Job Description**: Gunakan tombol + untuk menambah item, tombol - untuk menghapus
2. **Departemen**: Ketik nama departemen, sistem akan menyimpan dan menampilkan di filter
3. **Search**: Otomatis mencari di nama, posisi, telepon, dan email
4. **Export**: Filter data dulu sebelum export untuk hasil yang lebih spesifik

## Troubleshooting

**Q: Menu SDM tidak muncul?**  
A: Pastikan user Anda memiliki permission `hrm.karyawan.view`

**Q: Data tidak muncul?**  
A: Cek filter status dan departemen, pastikan tidak terlalu spesifik

**Q: Export tidak berfungsi?**  
A: Pastikan ada data yang ditampilkan di tabel
